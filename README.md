# 广财教务系统一键评教

## 简介
该项目由Awning Studio开发者开源，用于进行广财教务系统一键好评。Awning Studio还开源了另一个安卓软件[Afterglow](https://github.com/Awning-Studio/Afterglow)，欢迎你的使用。

## 用法
请确保你有Python环境
在命令行使用pip命令下载依赖库
```python
pip install requests
```
```python
pip install bs4
```
将Python文件保存到本地，并填写自己的学号密码
```python
USERNAME = "你的学号"
PASSWORD = "门户密码"
```
运行即可
